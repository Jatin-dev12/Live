# Template Preview Images

This directory contains preview images for page templates.

## Required Images:

1. **home-template-preview.png** - Preview of Home Template (Hero + 3 Callout Cards)
2. **about-template-preview.png** - Preview of About Template (Hero with Button + Tabs)
3. **legislation-template-preview.png** - Preview of Legislation Template (Hero with 2 Buttons + 1 Card)
4. **membership-template-preview.png** - Preview of Membership Template (Hero Section Only)

## Image Specifications:

- **Format**: PNG or JPG
- **Dimensions**: Recommended 400x300px or similar aspect ratio
- **Size**: Keep under 100KB for fast loading
- **Content**: Screenshot or mockup of the template layout

## How to Add Images:

1. Create screenshots of each template layout
2. Save them with the exact filenames listed above
3. Place them in this directory (`public/images/templates/`)
4. The preview images will automatically appear when users select templates
